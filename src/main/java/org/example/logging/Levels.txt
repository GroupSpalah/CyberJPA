TRACE (СЛІД): Найнижчий рівень. Зазвичай використовується для запису дуже деталей повідомлень,
корисних лише для налагодження.

DEBUG (НАЛАГОДЖЕННЯ): Використовується для виведення додаткової інформації
з метою налагодження. На цьому рівні можуть виводитися деталі виконання програми.

INFO (ІНФОРМАЦІЯ): Інформаційні повідомлення, які зазвичай використовуються
для повідомлення про хід виконання програми. Це може включати ключові моменти роботи програми.

WARN (ПОПЕРЕДЖЕННЯ): Використовується для повідомлень про можливі проблеми,
які не є критичними для роботи програми, але вимагають уваги.

ERROR (ПОМИЛКА): Використовується для повідомлень про помилки, які не
дозволяють програмі виконувати якусь дію. Це критичні проблеми, які вимагають уваги розробника.

FATAL (КРИТИЧНА ПОМИЛКА): Найвищий рівень. Використовується для повідомлень
про критичні помилки, які призводять до неможливості продовження виконання програми.
